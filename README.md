# AI Executive Assistant

An AI-powered executive operations assistant built using n8n, Gmail API, Telegram Bot API, and LLM workflows.

## Features

* Real-time Gmail monitoring
* AI-powered email prioritization
* Deadline extraction
* Action item detection
* Telegram executive alerts
* Automated professional email replies
* Workflow orchestration using n8n
* Secure API integration

## Workflow Architecture

Gmail Trigger → AI Analysis → Priority Classification → Telegram Alert → Gmail Auto Reply

## Tech Stack

* n8n
* OpenRouter / LLM APIs
* Gmail API
* Telegram Bot API
* Docker
* AI Workflow Automation

## Security

All API keys, OAuth secrets, tokens, and credentials have been removed before publishing.

## Future Improvements

* Slack integration
* Calendar event extraction
* Vector database memory
* React dashboard
